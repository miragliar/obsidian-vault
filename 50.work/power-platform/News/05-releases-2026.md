## Wichtigste Lessons aus den Releases (Q2 2026)

### 1. Microsoft öffnet sich für Drittanbieter-LLMs
- Mistral neben OpenAI in Copilot Studio (#266)
- Dataverse Plugin im Claude Marketplace (#263)
- **Implikation:** Multi-LLM-Strategien werden Beratungsthema. Kostenoptimierung, Compliance (in-region) als Achse.

### 2. Voice Agents werden produktreif
- CUA (Computer-Using Agents) ist GA (#266)
- Real-Time Voice ist GA mit Governance-Framework (#265)
- **Implikation:** Customer Service Use Cases können jetzt seriös abgedeckt werden.

### 3. ALM für Agents wird zur Disziplin
- James Papadimitriou (#267) ALM Foundation
- Vivian Voss Agent Evaluation (#267)
- Microsoft Copilot Studio Team Voice Governance (#265)
- **Implikation:** Wir brauchen explizite **Test/CI/CD-Patterns** für Copilot Studio Projekte. Test Sets, Promotion zwischen Environments, Monitoring.

### 4. MCP wird zum dominanten Integration-Standard
- Power Apps MCP, Power Pages MCP, Snowflake MCP, Dataverse MCP
- → Wenn ein neuer Datendienst integriert werden muss, **erst MCP-Pfad prüfen, dann Custom Connector**.

### 5. Power Fx als Foundation reift
- UDFs + UDTs GA in Q2 2026
- → Power Fx wird **produktionsreif** für komplexe Validierung (ParseJSON, strukturierte Records).
